# LiNa Skills

Diretório oficial para skills locais carregadas pelo `SkillLoader`.

## Estrutura

Cada skill deve viver em sua própria pasta:

`.agents/skills/<skill-name>/`

Arquivos canônicos:

- `MANIFEST.md`
- `INSTRUCTIONS.md`
- `CONTRACT.md`

## Campos esperados

- `name`
- `description`
- `version`
- `capabilities`
