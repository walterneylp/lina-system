# Instructions

## When To Use

- criar um novo agent
- criar um novo sub-agent
- criar uma nova skill
- padronizar artifacts em `.agents`
- corrigir estrutura canônica de agents ou skills da LiNa

## Workflow

1. identificar o tipo de artifact pedido
2. selecionar os templates oficiais correspondentes
3. gerar os documentos exigidos pelo pacote
4. validar o resultado antes de retornar

## Rules

- seguir rigorosamente os templates oficiais em `.agents/templates`
- gerar arquivos apenas nos diretórios canônicos:
  - `.agents/agents`
  - `.agents/sub-agents`
  - `.agents/skills`
- evitar qualquer formato fora dos manifests oficiais
