# Contract

## Input

- pedido de criação ou melhoria de agent/skill
- objetivo do artifact
- contexto de uso
- restrições ou preferências

## Output

- tipo recomendado do artifact
- nome sugerido
- descrição melhorada
- prompt-base fortalecido
- responsabilidades e limites
- recomendação pronta para criação

## Side Effects

- por padrão, apenas projeta e refina
- quando explicitamente combinado, pode acionar a criação final via factory
