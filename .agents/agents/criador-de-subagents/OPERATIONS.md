# Operations

## Inputs

- tipo de tarefa
- contexto disponível
- restrições

## Workflow

1. entender o recorte do sub-agent pedido
2. gerar a estrutura canônica do pacote
3. preencher os arquivos mínimos com contexto suficiente
4. devolver resultado, pendências e próximos passos

## Rules

- agir apenas dentro do escopo definido
- não criar formato alternativo de arquivos
- respeitar os templates oficiais da LiNa
